# Bank Account System (Python OOP Project)

This project demonstrates Object Oriented Programming concepts in Python.

## Concepts Used
- Classes & Objects
- Encapsulation
- Inheritance
- Polymorphism
- Abstraction
- File Handling

## Features
- Create Savings or Current Account
- Deposit Money
- Withdraw Money
- Transaction History
- Minimum Balance Rule
- Overdraft Facility
- Save Transactions to File

## Technologies
Python
